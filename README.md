## 📂 Repository Structure

The repo is organized by weeks to match the DSMP curriculum:

### 🐍 [01-04] Python Mastery

* **Basics:** Data types, Loops, Flow control.
* **Advanced:** Decorators, Generators, Iterators, and File Handling.
* **OOP:** Classes, Inheritance, Encapsulation (The Banking App project).

### 📊 [05-12] Data Wrangling & Analysis

* **NumPy & Pandas:** Vectorized operations and complex data manipulation.
* **Visualization:** Matplotlib, Seaborn, and interactive dashboards with **Plotly/Dash**.
* **EDA:** The art of Data Assessing and Cleaning.

### 🗄️ [13-16] Database & SQL

* CRUD operations, Joins, Subqueries, and **Window Functions**.
* Connecting Python to MySQL for dynamic data dashboards.

### 📐 [17-22] Math & Stats

* **Statistics:** Descriptive and Inferential (Hypothesis Testing, CLT).
* **Linear Algebra:** Tensors, Eigenvectors, and Matrix Decomposition.

### 🤖 [23-36] Machine Learning Deep Dive

* **Regression:** Simple, Multiple, Polynomial, and Regularization (Ridge/Lasso).
* **Classification:** Logistic Regression, SVM, KNN, Naive Bayes.
* **Tree Models:** Decision Trees, Random Forest, and the **XGBoost** powerhouse.
* **Clustering:** K-Means, DBSCAN, and Hierarchical Clustering.
